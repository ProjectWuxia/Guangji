# 水墨风动作插件

## 要求

1、操作在Photoshop CS5以上版本测试和运行

2、你的照片是在8位/通道 、RGB色彩模式

3、避免使用小分辨率图像。为了获得最佳效果，请使用3000px到6000px之间的照片

## 使用方法

1、在PS中载入abr笔刷文件（注：不透明度和流量必须100%）

2、打开要处理的照片

3、新建一空白层，命名为 `area`

4、在 `area` 图层上，选择合适的笔刷和颜色涂抹你要处理的图像区域

5、运行动作
